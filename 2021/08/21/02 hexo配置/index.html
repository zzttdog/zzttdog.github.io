001 Hexo

Hexo[1] 是最快速、简洁且高效的博客框架。

博客的搭建使用的 Hexo

Hexo 的安装

1、安装 Node.js
2、安装 Git
3、执行 npm install -g hexo -cli

Blog 的搭建

hexo init blog
cd blog
npm install

常用指令 Commands

hexo d -g

推荐网站

推荐一个快速上手的网站 easyhexo

主题我采用的是 Fluid

004 Valine

Valine 是一款快速、简洁且高效的无后端评论系统。[4]

评论区我用的是 Valine

Valine 的使用

1、注册 Leandcloud
2、在 Leandlound 的设置中找到 APP ID 和 APP KEY
3、在 config 中设置
4、npm install valine

005 Encrypt

Encrypt[5] 是 Hexo 生态圈中(最好的)博客加密插件。

部分文章使用了 Encrypt

Encrypt 的使用

1、npm install –save hexo-blog-encrypt

图表部分使用的 ECharts
