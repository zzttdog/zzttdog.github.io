Cyberpunk

Cyberpunk

简介

  赛博朋克( Cyberpunk )是“控制论”( Cybernetics )与“朋克”( Punk )的结合词，以信息技术为主体的科幻故事之分支。类似风格者还有以蒸气技术为主体的“蒸气朋克”、以生化技术为主体的“生化朋克”等等。[1]
  赛博朋克很大程度上诞生于1960至1970年代的新浪潮科幻运动。
劳仑斯·普尔森曾尝试定义赛博朋克文学运动的内涵和社会思潮：

  经典的赛博朋克角色是边缘且性格疏远的独行者。他们生活在社会群体的边缘，一个弥漫反乌托邦氛围的未来：日常生活受到急剧改变的科技影响，普及的计算机化信息笼罩全球，以及侵入性的人体改造。
  ——劳仑斯·普尔森

电影

银翼杀手 ( Blade Runner )

  《银翼杀手》改编自小说《仿生人会梦见电子羊吗？》，影片以其复杂的哲学思考和视觉效果而闻名，成为当代科幻电影的经典之作。它探索了人类与机器的道德和情感关系，同时也涉及了政治、社会和环境问题。银翼杀手的影响不仅在科幻电影领域广泛影响，还影响了其他艺术形式，如音乐、文学和游戏等。影片中的经典台词和场景也被引用和致敬无数次。

爱x死x机器人 ( Love, Death & Robots )

  《爱，死亡和机器人》（英语：Love, Death & Robots）是一部美国独立单元成人动画剧集，由乔舒亚·多南、大卫·芬奇、珍妮佛·米勒和蒂姆·米勒制作。每集由不同国家的团队所制作。第一季于2019年3月在奈飞首播，共18集。第二季于2021年5月释出，共8集。第三季于2022年5月释出，共9集。
  爱x死x机器人每集都采用了独特的动画风格和技术，涵盖了各种类型，其丰富且深刻的制作和天马行空的想象力令人着迷。
(据说2024年有第四季，期待！)

赛博朋克：边缘行者 ( Cyberpunk:Edgerunners )

  《赛博朋克：边缘行者》是由TRIGGER制作的一部网络动画，是游戏《赛博朋克2077》的衍生作品。该剧集共有10集，于2022年9月上线。
  Edgerunner 是原型桌游对玩家所扮演的角色的命名。其中 Edge 意喻角色活动区，而 Runner 则是来自著名赛博朋克题材作品银翼杀手( Blade Runner )。
  该动画把游戏赛博朋克2077带到了本不属于它的高度，给游戏业影视化的发展奠定了不小的契机，无论是画面还是音乐都是顶中顶，后劲十足。我愿称之为2022年度最佳独立动画！[4]

游戏

赛博朋克2077 ( Cyberpunk 2077 )

  《赛博朋克2077》（英语：Cyberpunk 2077，香港常用英文，台湾译作“电驭叛客2077”）是一款由CD Projekt开发并于2020年12月发行的动作角色扮演游戏。背景为2077年美国加利福尼亚州的“赛博朋克”风格反乌托邦式城市“夜之城”，玩家在这一开放世界中扮演大名鼎鼎的“ V ”。
  游戏最早在2012年以预告片形式公布，2018年3月21日开发商 CD Projekt Red 宣布在弗罗茨瓦夫开启一新的工作室协助游戏开发。在2019年E3电子娱乐展上宣布《赛博朋克2077》将于2020年4月16日发售，稍后公布游戏将登陆 Google Stadia 云游戏平台。2020年1月16日开发商 CD Projekt RED 宣布将游戏发行时间推迟到了2020年9月17日，2020年6月18日宣布延后至11月19日，2020年10月28日宣布再度延期至12月10日。
  “没有人会记得那些按时发布的垃圾游戏。”这句话出自 CD Projekt Red 的创始人马尔钦·伊温斯基（ Marcin Iwinski ）。该游戏公司以《巫师》系列而享誉天下，而在之后却又因为《赛博朋克2077》而臭名昭著。
（CDPR你真该给扳机社磕一个）

赛博朋克酒保行动 ( VA-11 HALL-A:Cyberpunk Bartender Action )

  《VA-11 Hall-A：赛博朋克酒保行动》（英语原名：VA-11 HALL-A:Cyberpunk Bartender Action）是一款关于waifu，科技及后-反乌托邦生活，以招待（灌醉）顾客为宗旨的调酒师模拟游戏。是一款具有视觉小说元素的独立调酒师模拟视频游戏，游戏由委内瑞拉工作室Sukeban Games开发，发行于2016年6月。
  赛博朋克酒保行动仅仅用一间小小的酒吧就描绘出了一个庞大的赛博朋克世界观。调制饮料，改变人生 ( Time to mix drinks and change life )。
  《VA-11 Hall-A：赛博朋克酒保行动》游戏制作者( P站ID：Kiririn51 )曾经立下flag表示游戏卖出10万份就亲自画主角的本子。期待该游戏的新作。(2020就已经放出预告了)[5]
